Research Projects


The current focus in my research group is on a new generation of "implicit" brain-computer interfaces. They allow a computer to obtain and act on auxiliary inputs from its user without requiring explicit user action or attention. Brain-computer interaction has made dramatic progress, but its main application to date has been for physically disabled users. Our work in real-time measurement and machine learning classification of functional near infrared spectroscopy (fNIRS) brain data is allows to create, use, and study new kinds of implicit user interfaces based on brain measurement.

We are using brain input as a way to obtain more information about the user and their context in an effortless and direct way from their brain activity. We then use it to adapt the user interface in real time. We are creating and studying these new user interfaces, with emphasis on domains where we can measure their efficacy.

Distinguishing Default Mode Network from Task Positive Network

Moving towards the development of implicit BCI for the average working individual, we are working to classify when a user's DMN or TPN is activated. This is a preliminary step to build better informed, adaptive user interfaces based on a user's neuro-phyisological states

Dynamically Adjusting VR Game Difficulty using combined fNIRS and EEG Data

As of 2025, there were over 3 billion active online gamers. Virtual environments provide an immersive experience that increase the engagement felt by a user. However, a gamer's performance does not have a linear relationship with the satisfaction and enjoyment a player feels while playing a video game and their enoyment is difficult to predict. Combined fNIRS and EEG can be used to monitor a player's performance in real time and be used as an input to adjust game difficulty. We are building a system with monitors a player's experience via brain signals and use the information as an input to the VR game to adjust the game difficulty and enhance the overall experience.
